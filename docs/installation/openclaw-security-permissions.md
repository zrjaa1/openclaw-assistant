# OpenClaw 安全权限配置指南

安装和初始配置完成后，强烈建议你在正式使用之前，先给小龙虾（OpenClaw）设置一个合适的权限级别。这一步非常重要 —— 它决定了小龙虾能在你的电脑上做哪些事情。

> 💡 好消息：从 2026.3.2 版本开始，新安装的小龙虾默认使用「消息模式」（messaging），只能收发消息，不能操作文件或执行命令。如果你是新安装的用户，你已经处于一个相对安全的状态。本指南帮你进一步了解和调整权限。
>
> ⚠️ 如果你是在 2026.3.2 之前安装的，或者手动修改过配置，你的小龙虾可能仍在使用「完全权限」模式。建议按照本指南检查和调整。

---

## 什么是权限级别？

小龙虾（OpenClaw）通过权限配置来控制 AI 助手能使用哪些功能。你可以把它想象成手机上的"应用权限" —— 你可以选择让它访问哪些功能，不让它访问哪些功能。

我们提供 3 个权限级别，从最安全到最开放：

---

### 1. 安全模式（Safe Mode）— 新手推荐

这是最安全的模式。小龙虾只能「看」，不能「动」你电脑上的任何东西。

能做的事：
- 阅读你电脑上的文件（只读，不能修改）
- 浏览互联网、搜索信息（需要先配置搜索功能）
- 通过聊天工具（微信、飞书、钉钉等）给你发送消息和报告
- 回忆之前记住的内容（读取记忆文件）
- 分析你发给它的图片和 PDF 文件

不能做的事：
- 不能创建、修改或删除任何文件
- 不能执行终端命令
- 不能控制浏览器
- 不能管理定时任务或设备

注意：在安全模式下，小龙虾无法保存新的记忆。也就是说，它不能把对话中学到的内容记录下来供以后使用。每次新对话都会从头开始（但已有的记忆文件仍然可以读取）。如果你觉得长期记忆功能很重要，可以考虑使用「读写模式」。

适合谁：刚开始使用小龙虾的新手，或者只需要一个聊天助手来回答问题、搜索信息的用户。

---

### 2. 读写模式（Read & Write Mode）— 进阶推荐

在安全模式的基础上，小龙虾还可以创建和修改文件。

能做的事（在安全模式基础上增加）：
- 创建新文件（比如帮你写报告、整理笔记）
- 修改文件内容
- 保存记忆（记住你告诉它的重要信息，下次对话还能想起来）
- 在它的工作目录下写代码

不能做的事：
- 不能执行终端命令（不能运行程序或脚本）
- 不能控制浏览器
- 不能管理定时任务或设备

注意：在这个模式下，小龙虾理论上可以写入电脑上的任何位置，不仅限于它的工作目录。但由于不能执行命令，它无法运行危险程序或脚本，风险大幅降低。

适合谁：希望小龙虾能记住事情、帮你整理文件或写代码的用户。

---

### 3. 完全权限（Full Mode）— 需要谨慎使用

小龙虾可以使用所有功能，没有任何限制。这是小龙虾的默认设置。

能做的事（在读写模式基础上增加）：
- 执行任何终端命令（运行程序、安装软件、执行脚本等）
- 控制浏览器（自动打开网页、填写表单、点击按钮等）
- 管理定时任务（设置定时提醒、自动化任务等）
- 控制连接的设备（如有配对的手机或其他电脑）
- 派生子任务给其他 AI 助手

⚠️ 注意：「完全权限」意味着小龙虾可以在你的电脑上执行任何操作，包括运行任何程序、安装或删除软件等。这是默认设置，但除非你完全了解这些权限的含义，否则不建议使用此模式。

适合谁：完全信任小龙虾，且清楚了解所有功能含义的高级用户或开发者。

## 如何设置权限？

### 方法一：用命令设置（推荐）

打开终端（Windows 用 PowerShell），复制并运行下面的命令：

安全模式（推荐新手）：

openclaw config set tools.deny '["write","edit","apply_patch","exec","process","sessions_spawn","nodes","canvas","cron","gateway"]'

读写模式（推荐进阶用户）：

openclaw config set tools.deny '["exec","process","sessions_spawn","nodes","canvas","cron","gateway"]'

完全权限（高级用户，慎用）：

openclaw config set tools.deny '[]'

设置完后，重启小龙虾让设置生效：

openclaw gateway restart

### 方法二：编辑配置文件

1. 找到你的小龙虾配置文件：

Windows:
%USERPROFILE%\.openclaw\openclaw.json

Mac/Linux:
~/.openclaw/openclaw.json

2. 用文本编辑器打开，找到或添加 "tools" 部分：

安全模式：

{
  "tools": {
    "deny": ["write", "edit", "apply_patch", "exec", "process", "sessions_spawn", "nodes", "canvas", "cron", "gateway"]
  }
}

读写模式：

{
  "tools": {
    "deny": ["exec", "process", "sessions_spawn", "nodes", "canvas", "cron", "gateway"]
  }
}

完全权限：

{
  "tools": {
    "deny": []
  }
}

3. 保存文件，然后重启：

openclaw gateway restart

## 权限级别对比速查

安全模式 — 只读，最安全，不能保存记忆（新手推荐）
读写模式 — 可以读写文件和保存记忆，不能执行命令（进阶推荐）
完全权限 — 没有限制，可以做任何事（默认，谨慎使用）

## 常见问题

Q：设置了权限后，以后想改怎么办？
A：随时可以重新运行上面的命令来更改权限级别。改完后记得运行 openclaw gateway restart。

Q：我选了安全模式但后来想让它记住东西怎么办？
A：把权限改成读写模式就行，参考上面的设置方法。

Q：为什么默认是完全权限？不是很不安全吗？
A：默认完全权限是为了让高级用户开箱即用。但对于大多数用户来说，安全模式或读写模式已经足够，而且更安全。所以我们建议你在开始使用前先设置一个合适的权限。

Q：搜索功能需要单独配置吗？
A：是的，网页搜索功能需要配置一个搜索服务的 API Key（比如 Brave Search 或 Perplexity）。不配置也可以使用小龙虾，只是不能搜索网页。小龙虾仍然可以通过 web_fetch 访问你给它的具体网址。
