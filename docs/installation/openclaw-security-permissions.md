# OpenClaw 安全权限配置指南

安装和初始配置完成后，建议你了解小龙虾（OpenClaw）的权限级别，根据自己的需要选择合适的模式。

> 💡 好消息：从 2026.3.2 版本开始，新安装的小龙虾默认使用「安全模式」，只能收发消息，不能操作文件或执行命令。如果你是新安装的用户，你已经处于最安全的状态。
>
> ⚠️ 如果你是在 2026.3.2 之前安装的，或者手动修改过配置，你的小龙虾可能仍在使用「完全权限」模式。建议按照本指南检查和调整。

---

## 4 个权限级别

从最安全到最开放：

---

### 1. 安全模式（Safe Mode）— 默认，新手推荐

这是新安装（2026.3.2+）的默认模式。小龙虾只能收发消息，什么都不能动。

能做的事：
- 通过聊天工具（微信、飞书、钉钉等）收发消息
- 查看对话历史
- 查看运行状态

不能做的事：
- 不能阅读你电脑上的文件
- 不能浏览互联网或搜索信息
- 不能分析图片或 PDF
- 不能创建、修改或删除文件
- 不能执行终端命令
- 不能保存长期记忆

适合谁：刚开始使用小龙虾的新手，或者只需要一个简单的聊天助手。

如果你是新安装的用户，不需要做任何设置，这就是默认状态。

---

### 2. 只读模式（Read-Only Mode）

在安全模式的基础上，小龙虾可以「看」更多东西，但仍然不能「动」。

能做的事（在安全模式基础上增加）：
- 阅读你电脑上的文件（只读，不能修改）
- 浏览互联网、搜索信息（需要先配置搜索功能）
- 分析你发给它的图片和 PDF 文件
- 回忆之前记住的内容（读取记忆文件）
- 管理对话和子任务

不能做的事：
- 不能创建、修改或删除任何文件
- 不能执行终端命令
- 不能保存新的记忆（无法写入文件）

注意：只读模式下小龙虾无法保存新的记忆。每次新对话都会从头开始（但已有的记忆文件仍然可以读取）。如果你觉得长期记忆功能很重要，可以考虑使用「读写模式」。

适合谁：希望小龙虾能帮你查资料、读文件、分析内容，但不希望它修改任何东西的用户。

设置方法：

复制下面这行命令粘贴到终端里（Windows 用 PowerShell）：

openclaw config set tools.deny '["write","edit","apply_patch","exec","process","sessions_spawn","nodes","canvas","cron","gateway"]'

然后重启：

openclaw gateway restart

---

### 3. 读写模式（Read & Write Mode）— 进阶推荐

在只读模式的基础上，小龙虾还可以创建和修改文件。

能做的事（在只读模式基础上增加）：
- 创建新文件（比如帮你写报告、整理笔记）
- 修改文件内容
- 保存长期记忆（记住你告诉它的重要信息，下次对话还能想起来）
- 在它的工作目录下写代码

不能做的事：
- 不能执行终端命令（不能运行程序或脚本）
- 不能控制浏览器
- 不能管理定时任务或设备

适合谁：希望小龙虾能记住事情、帮你整理文件或写代码的用户。

设置方法：

openclaw config set tools.deny '["exec","process","sessions_spawn","nodes","canvas","cron","gateway"]'

然后重启：

openclaw gateway restart

---

### 4. 完全权限（Full Mode）— 需要谨慎使用

小龙虾可以使用所有功能，没有任何限制。老版本（2026.3.2 之前）的默认设置。

能做的事（在读写模式基础上增加）：
- 执行任何终端命令（运行程序、安装软件、执行脚本等）
- 控制浏览器（自动打开网页、填写表单、点击按钮等）
- 管理定时任务（设置定时提醒、自动化任务等）
- 控制连接的设备（如有配对的手机或其他电脑）
- 派生子任务给其他 AI 助手

⚠️ 注意：「完全权限」意味着小龙虾可以在你的电脑上执行任何操作，包括运行任何程序、安装或删除软件等。除非你完全了解这些权限的含义，否则不建议使用此模式。

适合谁：完全信任小龙虾，且清楚了解所有功能含义的高级用户或开发者。

设置方法：

openclaw config set tools.deny '[]'

然后重启：

openclaw gateway restart

---

## 权限级别对比速查

安全模式 — 只能收发消息（默认，新手推荐）
只读模式 — 还可以读文件、搜索网页、分析内容（不能保存记忆）
读写模式 — 还可以写文件、保存记忆（进阶推荐）
完全权限 — 没有限制，可以做任何事（慎用）

## 常见问题

Q：我是新安装的用户，需要做什么吗？
A：新安装（2026.3.2+）默认就是安全模式，不需要做任何设置。如果你想要更多功能，可以按上面的说明升级到只读或读写模式。

Q：设置了权限后，以后想改怎么办？
A：随时可以重新运行上面的命令来更改权限级别。改完后记得运行 openclaw gateway restart。

Q：我选了只读模式但后来想让它记住东西怎么办？
A：把权限改成读写模式就行，参考上面的设置方法。

Q：怎么检查我当前是什么权限？
A：运行 openclaw status 可以查看当前的配置状态。

Q：搜索功能需要单独配置吗？
A：是的，网页搜索功能需要配置一个搜索服务的 API Key（比如 Brave Search 或 Perplexity）。不配置也可以使用小龙虾，只是不能主动搜索网页。小龙虾仍然可以通过 web_fetch 访问你给它的具体网址。
